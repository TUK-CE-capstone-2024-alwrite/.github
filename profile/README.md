# 📺 노트 필기 어플리케이션 Alwrite

## 💡프로젝트 소개
  -OCR 기능을 활용한 다목적 필기 어플리케이션입니다.
  -저희 어플은 다양한 필기 기능을 제공합니다. 
  -텍스트를 내가 원하는 폰트로 바꿔주는 텍스트 변환 기능과 녹음을 요약하여 텍스트로 보여주는 기능을 추가하여
  -기존 어플리케이션과의 차별점을 두었습니다.
